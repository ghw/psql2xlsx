Postgres SQL to .xlsx
======================

Utility for saving Postgres SQL querys results to .xlsx file 


Required
=======
* Python 3.x
  * psycopg2
  * xlsxwriter


Example
=======
Help
```console
user@localhost:~$ ./psql2xlsx.py -h
```

Visit also [WiKi](http://wiki.enchtex.info/handmade/postgres/psql2xlsx).
